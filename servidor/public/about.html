Objetivo General

Controlar el acceso de personas a un inmueble través de cerraduras electrónicas.

Objetivos Específicos

  • Crear una base de datos que almacene los datos de las cerraduras, de los usuarios y de los eventos realizados en cada cerradura.
  • Crear un servidor Node.js para gestión de datos y nodos.
  • Diseñar una interfaz web para administrar el sistema.
  • Construir un prototipo de cerradura con dispositivos de entrada y salida para el usuario.
  • Conectar el sistema a una red LAN inalámbrica.
  • Realizar pruebas para evaluar el funcionamiento del sistema.

Justificación

Los edificios de oficinas, que generalmente tienen un alto tráfico de personas y vehículos, se favorecen ampliamente del control de acceso. Un sistema de control de acceso con Plataforma WEB le permitirá confirmar en todo momento quién solicitó ingresar y realizar autorizaciones de ingreso de forma sencilla y fácil.

Los controles de acceso también tienen aplicación en las unidades colectivas de vivienda, como áreas de copropiedad o condominios campestres. Uno de los atributos de ese tipo de propiedades es esencialmente que ofrecen un mayor nivel de seguridad. Esta se logra garantizando que quienes circulan por los espacios compartidos están autorizados para hacerlo.

Lo mismo ocurre en las zonas francas, centros comerciales, parques industriales, almacenes, centros médicos, entre otros. Conociendo que este tipo de espacios suelen tener áreas restringidas en su interior y, por lo mismo, demandan un control más detallado. Los controles de acceso no simplemente monitorean el acceso general, sino que también ofrecen control sobre zonas o dependencias específicas.

Alcance

El sistema de control de acceso cuenta con una interfaz web para administrar las restricciones por usuario, así a cada usuario registrado con una contraseña el administrador le otorgará permiso para abrir una o más puertas.

Cada puerta estará vinculada a un departamento en específico, también se registrarán eventos que contendrán datos de la puerta, del usuario y el suceso en específico que ocurrió con la fecha del mismo, por ejemplo: Pedro abrió la puerta P1 ubicada en el Laboratorio de electricidad el 12/09/17 10:25:09.

La cerradura electrónica contará con una membrana de teclado matricial para ingreso de datos de autenticación de usuario, además contará con un LCD 16x2 hacer posible la visualización de los datos ingresados y para mostrar el mensaje de respuesta a la solicitud según corresponda.

Cada nodo puerta se comunicará de forma inalámbrica utilizando la tecnología WiFi, el sistema de cierre y apertura se realizará con el suministro y corte de energía respectivamente.
